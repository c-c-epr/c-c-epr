## ⚡關於我

從小就對電子產品有很大的興趣，最開始是對 iPad 調整各種設定(沒到越獄那麼厲害)。曾有一陣子熱衷於寫捷徑(Shortcuts)。

由於很多原因，我第一個學會的「程式語言」勉強算是是 Scratch。後來才學了 Python，現在則是在學 JavaScript。

不是很會寫程式，目標也不是成為技術非常頂尖的工程師。更喜歡和擅長用「工具」解決問題，這也是我很喜歡 Shortcuts 和 n8n workflow 的原因。

目前在 GitHub 上，除了維護自己的專案，也熱衷於幫好用的專案加上繁體中文翻譯。但品質很難說，畢竟我學中文也才10幾年。若是有開源專案需要翻譯，不嫌棄的話可以找我。
